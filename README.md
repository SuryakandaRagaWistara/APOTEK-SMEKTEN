# Pharmacy Inventory & Transaction Management System

A system designed to manage pharmacy inventory, suppliers, transactions, and financial records efficiently.  
This project was built based on real field observations to solve common operational problems in pharmacy inventory management.

## System Overview

Pharmacies often face several challenges such as managing medicines with multiple unit types, monitoring expiration dates, comparing supplier prices, and handling flexible payment transactions.

This system provides structured tools to help manage inventory, reduce risk, and track financial transactions in a more organized way.

---

## Main Features

### 1. Inventory & Vendor Management

**Multi-Unit Support**  
The system allows medicine prices to be recorded in multiple unit levels such as:

- Box
- Strip
- Bottle
- Individual item (smallest unit)

This helps pharmacies manage medicines that are sold in different packaging levels.

**Medical Dosage Tracking**  
The system supports recording medicine dosage using gram-based measurements to ensure accurate medical documentation.

**Supplier Database**  
Supports managing multiple suppliers and generates separate transaction reports for each vendor.

**Vendor Discount System**  
A dedicated field is provided to record supplier discounts, helping determine competitive selling prices.

---

### 2. Quality Control & Risk Management

**Expiration Date Monitoring**  
The system provides early warnings for medicines approaching expiration within **3–6 months**.

**Loss Analysis**  
The system generates reports for damaged or expired medicines to help track potential financial losses.

**Vendor Price Comparison**  
Allows comparison of medicine prices from multiple suppliers to support better purchasing decisions.

---

### 3. Transaction & Financial System

**Flexible Payment Methods**

The system supports several types of transactions, including:

- Cash payments
- Installment payments
- Debt / credit transactions (for example, student receivables)

**Cash Flow Reporting**

Provides daily and monthly financial summaries including:

- Incoming cash
- Payments to suppliers
- Remaining inventory value
- Profit and loss overview

**Receivable Management**

The system tracks customers or students who make partial payments or installment-based transactions.

---

## Project Purpose

This system was created to improve pharmacy operational efficiency by providing:

- Structured inventory management  
- Clear financial tracking  
- Supplier price optimization
